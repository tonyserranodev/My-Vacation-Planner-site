D424 Capstone "My Vacation Planner"

My Vacation Planner is an Android mobile app developed for the WGU D424 Software Engineering Capstone. This application helps users track and organize their vacations and excursions in one convenient place.

Features

  • Create, edit, and view vacation details.
  • Manage excursions associated with each vacation.
  • Set reminders for upcoming events.
  • Share vacation details via SMS or email.
